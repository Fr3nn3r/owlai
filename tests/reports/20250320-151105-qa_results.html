Il sert de base pour la plupart des conventions bilatérales et couvre des aspects tels que la résidence fiscale, les types de revenus, et les méthodes pour éviter la double imposition. 2. **Modèle de convention des Nations Unies :** - Ce modèle est similaire à celui de l'OCDE mais est souvent utilisé par les pays en développement. Il accorde généralement plus de droits de taxation au pays source des revenus, par rapport au pays de résidence. ### Mécanismes pour éviter la double imposition : 1. **Exemption :** - Un pays peut choisir d'exempter de l'impôt les revenus qui ont déjà été imposés dans l'autre pays contractant. Cela signifie que le revenu n'est imposé que dans le pays où il est généré. 2. **Crédit d'impôt :** - Le pays de résidence de la personne ou de l'entité peut accorder un crédit d'impôt pour les impôts payés dans le pays source. Ce crédit est généralement égal à l'impôt payé à l'étranger, mais il est limité à l'impôt qui aurait été dû dans le pays de résidence. 3. **Réduction des taux d'imposition :** - Les conventions peuvent prévoir des taux d'imposition réduits pour certains types de revenus, tels que les dividendes, les intérêts, et les redevances, afin de limiter la charge fiscale dans le pays source. 4. **Règles de résidence fiscale :** - Les conventions définissent des critères pour déterminer la résidence fiscale des contribuables, ce qui aide à clarifier quel pays a le droit d'imposer certains types de revenus. ### Raisonnement : Les conventions fiscales internationales sont essentielles pour faciliter le commerce et l'investissement transfrontaliers en éliminant les obstacles fiscaux.
